# ORBIT Roadmap

## Phase 1 (Completed)
- Local tracing engine via `@trace_agent`
- Ollama local model integration
- React UI Dashboard
- Basic Security Guard (Prompt Injection, Llama Guard 3)
- SQLite database logging

## Phase 2 (Planned)
- Expand to non-LangGraph frameworks (CrewAI, AutoGen)
- Provide `--live` step-by-step UI replay execution
- Advanced Agent Arena testing datasets

## Phase 3 (Future)
- PII Redaction
- Advanced benchmark suites for code generation
